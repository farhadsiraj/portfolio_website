reUnion(Work In Progress)

Plan. Invite. re-Unite.

Github

reUnion is an event planning app that allows users to plan and manage their events. Users can sign up for an account, create an event, plan activities, and invite other users to their events.

reUnion is a project that a few of my fellow graduates and I created to practice the skills that we learned at Fullstack Academy. The goal is to practice the tech stack that we already know and branch out and learn new technologies. A few of the things Iâ€™ve learned so far are how to use React Hooks, write unit tests in Jest, create swipeable components using SwiperJS, and create custom animations using vanilla CSS.

In the near future, we plan on migrating our backend database to AWS, implementing authentication using Amazon Cognito, creating a chat feature using Socket.IO, and building out a marketplace.
